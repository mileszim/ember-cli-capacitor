ember-cli-capacitor
==============================================================================

This addon integrates Ember and Capacitor.


Compatibility
------------------------------------------------------------------------------

* Ember.js v3.12 or above
* Ember CLI v2.13 or above
* Node.js v10 or above


Installation
------------------------------------------------------------------------------

```
ember install ember-cli-capacitor
```


Usage
------------------------------------------------------------------------------

### Commands

```
ember ios:serve
```

```
ember ios:build
```
